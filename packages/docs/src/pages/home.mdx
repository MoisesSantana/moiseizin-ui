# Instalação

## Como instalar o Design System do Moiseizin Dev

<br />
<br />

### Instalando MoiseizinUI
Para instalar o MoiseizinUI, você pode usar o `npm` ou `yarn`.

```bash
# Com npm
npm install @moiseizin/ui
```

```bash
# Com yarn
yarn add @moiseizin/ui
```
